play_on_xbmc
======

# Description:
Short script to send YouTube videos to an XBMC instance.

# Run without video ID:
<pre>
# ./play_on_xbmc.py
What is the ID of the YouTube video?
> dQw4w9WgXcQ
Playing 'Rick Astley - Never Gonna Give You Up'
</pre>

# Specify video ID:
<pre>
# ./play_on_xbmc.py dQw4w9WgXcQ
Playing 'Rick Astley - Never Gonna Give You Up'
</pre>
